# Go-WebServer
## A sample Web-Server using Golang and HTML.
<p>
  Steps:
  <br>
  1) Start server at port 8080 using commands: 
  
```bash
  go build 
  go run main.go
```
  <br>
  2) We will be landing at the starting page.
  <br>
  3) Use commands:

```bash
  /hello
  /form.html
```
  to open the respective pages.
</p>
